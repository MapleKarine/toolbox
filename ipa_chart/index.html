Usage

; example phonology
add voiced bilabial stop "ᵐb"
add [d] "ⁿd"
add front "e"
add back "o"
p t k s h m n w l
a i u

The simplest way to use the tool is to paste vowels and consonants in the text box and let it be automatically organized. A finer control of the sounds can be made using the add command.

The list of phonemes must be the very last thing in the input.

The add command can be followed by a series of keywords voiced bilabial stop or an IPA segment written in square brackets [d] that signifies its position, and a display text, written in quotes "ᵐb".

A line starting with ; is ignored.

Vowel Position

Vowels can also be positioned by numerical coordinates, the coordinate (0, 0) corresponds with the [i] position, and (2, 3) with the [ɑ] position. Decimal numbers like (1.5, 1.25) can be used.

Settings

Settings can be changed using the visual interface or with the config <config-name> value command.

The value can be true,false or a double quoted string "option".

; add a row for alveolar sibilants [s ts dz ...]
config sibilant true

; merge stops and affricates in the same row
config affricate true

; merge trills, taps, approximants and laterals in the same row
config sonorant true

; move nasals to the top, taps and trills to the bottom
config nasalTop true

; arrange MOA the same order as the oficial IPA chart
config nasalTop false

; add automatic columns for labialized and palatalized sounds with ʷʲ
config labialized true
